Customer Churn Analysis Dashboard

A visually rich and interactive Customer Churn Analysis Dashboard built to help businesses understand customer behavior, identify churn patterns, and improve retention strategies.

Purpose:
This project presents a comprehensive dashboard for analyzing customer churn using key metrics such as total customers, churn rate, retention distribution, contract trends, tenure impact, and service usage.
It enables clear visibility into customer behavior and churn patterns.
The goal is to transform raw data into actionable insights that support better retention strategies.

Tech Stack:
Power BI Desktop – Developed interactive dashboards and visual reports.
Power Query – Performed data extraction, cleaning, and transformation (ETL).
Data Modeling – Designed relationships, schemas, and optimized data structures for analysis.
DAX (Data Analysis Expressions) – Created calculated measures, columns, and KPIs.
File Format – .pbix for report development and distribution and .png for dashboard previews.

Data Source:
The dataset for this project was obtained from a publicly shared dataset referenced in a YouTube tutorial description and is used here for educational and analytical purposes.

Business Problem:
Businesses face high customer churn, particularly among new or high-risk customers, resulting in revenue loss and ineffective retention efforts. There is limited visibility into which factors—such as contract type, tenure, or service usage—drive churn.

Goal of the Dashboard:
To analyze customer churn, uncover patterns in behavior, and provide actionable insights that help reduce churn and improve customer retention strategies.

Key Visuals:
KPI Cards: Total customers, churn rate, churned customers, retention rate.
Contract-Based Churn: Visualizes churn trends by contract type (Month-to-Month, 1-Year, 2-Year).
Tenure Analysis: Shows churn distribution across customer tenure bands (0–6 months, 6–12 months, 12+ months).
Service Adoption Impact: Examines how services like Online Security and Backup influence churn.
Retention Distribution: Highlights high-risk vs. normal customers.
Revenue Analysis: Compares revenue contributions of retained vs. churned customers.

Business Impact and Insights:
Helps businesses proactively identify high-risk customers.
Highlights the key factors driving churn.
Month-to-Month contracts and customers in their first 0–6 months are most likely to churn.
Lack of value-added services (Online Security, Backup) further increases churn risk.
Enables businesses to design targeted retention campaigns.
Encourages promotion of long-term contracts to improve customer loyalty.
Supports improved service adoption strategies.
Ultimately helps reduce churn while maximizing revenue and customer lifetime value.
